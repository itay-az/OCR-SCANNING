# PDF Rename Tool - כלי שינוי שמות קבצי PDF

אפליקציית Windows desktop לעיבוד קבצי PDF, חיפוש ערכים באמצעות REGEX, ושינוי שמות קבצים אוטומטי.

## תכונות

- ✅ עיבוד קבצי PDF searchable (עם טקסט)
- ✅ תמיכה ב-OCR לקבצים סרוקים (דורש Tesseract)
- ✅ חיפוש REGEX מותאם אישית
- ✅ שינוי שמות קבצים אוטומטי
- ✅ ממשק משתמש נוח בעברית
- ✅ לוג מפורט של כל הפעולות

## דרישות מערכת

- Windows 10/11
- Python 3.8 או גרסה חדשה יותר
- Tesseract OCR (אופציונלי - רק אם רוצים תמיכה ב-OCR)

## התקנה

### 1. התקנת Python

אם Python לא מותקן במחשב שלך:
1. הורד Python מ-[python.org](https://www.python.org/downloads/)
2. במהלך ההתקנה, ודא שסימנת את "Add Python to PATH"
3. התקן Python

### 2. התקנת Tesseract OCR (אופציונלי)

**חשוב:** Tesseract נדרש רק אם אתה רוצה לעבד קבצי PDF סרוקים (ללא טקסט).

1. הורד את Tesseract מ-[UB Mannheim Tesseract](https://github.com/UB-Mannheim/tesseract/wiki)
   - בחר את הגרסה המתאימה ל-Windows (64-bit מומלץ)
2. התקן את Tesseract
3. במהלך ההתקנה, ודא שסימנת:
   - ✅ "Add to PATH" (או הוסף ידנית ל-PATH)
   - ✅ Hebrew language data (לתמיכה בעברית)
4. לאחר ההתקנה, בדוק שההתקנה הצליחה:
   ```cmd
   tesseract --version
   ```

### 3. התקנת התלויות

1. פתח Command Prompt או PowerShell
2. נווט לתיקיית הפרויקט:
   ```cmd
   cd "C:\dev\OCR PROJECT"
   ```
3. התקן את התלויות:
   ```cmd
   pip install -r requirements.txt
   ```

## שימוש

### הפעלת האפליקציה

```cmd
python main.py
```

### שימוש באפליקציה

1. **בחר תיקייה**: לחץ על "בחר תיקייה" ובחר תיקייה המכילה קבצי PDF
2. **הזן REGEX**: הזן תבנית REGEX לחיפוש הערך שברצונך להשתמש כשם קובץ
   - דוגמה: `\d{9}` - מחפש 9 ספרות רצופות (תעודת זהות)
   - דוגמה: `[0-9]{8,9}` - מחפש 8-9 ספרות
   - דוגמה: `ID:\s*(\d+)` - מחפש "ID:" ואחריו מספר
3. **הרץ עיבוד**: לחץ על "הרץ עיבוד"
4. **בדוק את התוצאות**: האפליקציה תציג לוג מפורט של כל הפעולות

### דוגמאות REGEX

| תבנית | תיאור | דוגמה התאמה |
|-------|-------|-------------|
| `\d{9}` | בדיוק 9 ספרות | `205612955` |
| `\d{8,9}` | 8 או 9 ספרות | `20561295`, `205612955` |
| `[0-9]+` | כל רצף ספרות | `123`, `456789` |
| `ID:\s*(\d+)` | "ID:" ואחריו מספר | `ID: 205612955` |
| `תעודת זהות[:\s]+(\d+)` | "תעודת זהות" ואחריו מספר | `תעודת זהות: 205612955` |

**טיפ:** השתמש באתרים כמו [regex101.com](https://regex101.com) לבדיקת תבניות REGEX שלך.

## איך זה עובד

1. **PDFs Searchable**: האפליקציה קוראת את הטקסט ישירות מהקובץ
2. **PDFs סרוקים**: אם לא נמצא טקסט, האפליקציה:
   - ממירה את ה-PDF לתמונות
   - מבצעת OCR באמצעות Tesseract
   - מחפשת את ה-REGEX בטקסט שנחלץ
3. **שינוי שם**: אם נמצאה התאמה, הקובץ משנה את שמו לערך שנמצא

## פתרון בעיות

### Tesseract לא נמצא

אם אתה מקבל הודעה ש-Tesseract לא נמצא:
1. ודא ש-Tesseract מותקן
2. ודא ש-Tesseract ב-PATH:
   ```cmd
   tesseract --version
   ```
3. אם זה לא עובד, הוסף ידנית את נתיב Tesseract ל-PATH:
   - בדרך כלל: `C:\Program Files\Tesseract-OCR`

### שגיאת הרשאות

אם אתה מקבל שגיאת הרשאות:
- ודא שהקובץ לא פתוח בתוכנה אחרת (כמו Adobe Reader)
- ודא שיש לך הרשאות כתיבה לתיקייה

### קבצים לא משתנים

אם קבצים לא משנים את שמם:
- בדוק שהתבנית REGEX נכונה
- בדוק בלוג אם נמצאה התאמה
- ודא שהקובץ הוא PDF תקין

### שגיאת התקנה

אם יש בעיות בהתקנת חבילות Python:
```cmd
pip install --upgrade pip
pip install -r requirements.txt
```
